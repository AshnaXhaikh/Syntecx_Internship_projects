# Pandas CSV Reader & Basic Analysis Tool

A comprehensive data analysis tool demonstrating pandas fundamentals for data exploration and manipulation.

## 🎯 Features

### Core Capabilities
- ✅ Read CSV/Excel files into DataFrames
- ✅ Data inspection (head, tail, info, describe)
- ✅ Summary statistics (mean, median, min, max, count, std)
- ✅ Advanced filtering and querying
- ✅ Column selection and subset creation
- ✅ Save results to CSV/Excel

### Advanced Features
- 📊 Grouped statistics and pivot tables
- 🔍 Custom filtering functions
- 📈 Time-based analysis
- 💾 Multi-sheet Excel exports
- 📋 Data quality reporting
- 🎨 Interactive filtering interface

## 📊 Sample Dataset

The project generates a realistic sales dataset with:
- **1,000 orders** spanning 1 year
- **5 product categories** (Laptop, Phone, Tablet, Headphones, Charger)
- **Customer segmentation** (New, Regular, VIP)
- **Geographic distribution** (5 major US cities)
- **Discount analysis** (0-20% range)
