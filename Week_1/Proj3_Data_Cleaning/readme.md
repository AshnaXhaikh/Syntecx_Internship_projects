# Data Cleaning Utility

A professional, production-ready data cleaning toolkit that handles common data quality issues automatically.

## 🎯 Features

### Core Cleaning Operations
- ✅ **Standardize column names** (lowercase, underscores, no special chars)
- ✅ **Fix incorrect data types** (dates, numbers, categories)
- ✅ **Handle missing values** (mean/median/mode filling)
- ✅ **Remove duplicates** with configurable subset
- ✅ **Clean text columns** (strip whitespace, validate emails/phones)
- ✅ **Handle outliers** (IQR method with capping)

### Advanced Features
- 📋 **Comprehensive cleaning log** with timestamps
- 📊 **HTML cleaning report** generation
- 🔄 **Batch processing** for multiple files
- 🎯 **Configurable strategies** for missing values
- ✅ **Data validation** after cleaning
- 📈 **Before/after comparison metrics
